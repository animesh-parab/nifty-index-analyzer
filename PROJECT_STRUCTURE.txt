🤖 AI PREDICTION ENGINE
-----------------------

ai_engine_consensus.py (500+ lines)
- Dual-AI consensus system
- Groq (Llama 3.3 70B) + Gemini (2.5 Flash)
- Parallel API calls (3-5 seconds)
- Consensus calculation:
  * STRONG: Both agree → Confidence +10%
  * MODERATE: Partial agreement → Confidence -5%
  * WEAK: Disagreement → Confidence -20%
- Fallback to rule-based if both fail
- Functions:
  * get_consensus_prediction()
  * get_rule_based_prediction()
  * _call_groq_api()
  * _call_gemini_api()
  * _calculate_consensus()
