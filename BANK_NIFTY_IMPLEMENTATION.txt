HOW IT WORKS
================================================================================

USER FLOW:
1. User opens dashboard
2. Sees dropdown with "NIFTY" and "BANKNIFTY" options
3. Selects desired index
4. Dashboard fetches data ONLY for selected index
5. All features work for selected index:
   - Real-time price
   - Technical indicators
   - AI predictions
   - Options chain
   - Charts
   - Price alerts

TECHNICAL FLOW:
1. User selects index from dropdown
2. selected_index variable updated
3. cached_price(selected_index) called
4. cached_candles(selected_index) called
5. Data fetchers use index parameter
6. API calls made only for selected index
7. Dashboard displays selected index data

SUPPORTED DATA SOURCES
================================================================================

All data sources support both indices:

1. NSE API
   - NIFTY 50: https://www.nseindia.com/api/equity-stockIndices?index=NIFTY%2050
   - BANK NIFTY: https://www.nseindia.com/api/equity-stockIndices?index=NIFTY%20BANK

2. Angel One
   - NIFTY 50: Token 99926000
   - BANK NIFTY: Token 99926009

3. yfinance
   - NIFTY 50: ^NSEI
   - BANK NIFTY: ^NSEBANK
