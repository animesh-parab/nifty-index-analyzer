  CONSENSUS CALCULATION LOGIC
═══════════════════════════════════════════════════════════════════════════

SCENARIO 1: All Agree (STRONG Consensus)
─────────────────────────────────────────
XGBoost:  BULLISH (68%)
Groq:     BULLISH (72%)
Gemini:   BULLISH (65%)
                ↓
Weighted: (68*0.6 + 72*0.2 + 65*0.2) = 68.2%
Bonus:    +10% (all agree)
Final:    BULLISH (78%) - STRONG consensus


SCENARIO 2: XGB Agrees with Final (MODERATE Consensus)
───────────────────────────────────────────────────────
XGBoost:  BULLISH (65%)
Groq:     BULLISH (70%)
Gemini:   SIDEWAYS (55%)
                ↓
Weighted: (65*0.6 + 70*0.2 + 55*0.2) = 64%
Bonus:    0% (partial agreement)
Final:    BULLISH (64%) - MODERATE consensus


SCENARIO 3: XGB Disagrees (WEAK Consensus)
───────────────────────────────────────────
XGBoost:  BEARISH (62%)
Groq:     BULLISH (75%)
Gemini:   BULLISH (68%)
                ↓
Weighted: (62*0.6 + 75*0.2 + 68*0.2) = 65.8%
Penalty:  -15% (XGB disagrees)
Final:    BULLISH (51%) - WEAK consensus


SCENARIO 4: XGBoost Only (No LLMs)
───────────────────────────────────
XGBoost:  BULLISH (70%)
Groq:     ERROR
Gemini:   ERROR
                ↓
Final:    BULLISH (70%) - XGB_ONLY


SCENARIO 5: Dual-AI Only (No XGBoost)
──────────────────────────────────────
XGBoost:  Not trained yet
Groq:     BULLISH (72%)
Gemini:   BULLISH (68%)
                ↓
Weighted: (72*0.5 + 68*0.5) = 70%
Final:    BULLISH (70%) - DUAL_AI consensus
