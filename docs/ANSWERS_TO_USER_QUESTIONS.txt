QUESTION 1: If Angel One and NSE both fail, does the app crash or just
            pause predictions cleanly?
───────────────────────────────────────────────────────────────────────

ANSWER: App DOES NOT CRASH - Pauses predictions cleanly ✅

How it works:
1. NSE API fails → Try Angel One
2. Angel One fails → Try yfinance (15 min delay)
3. yfinance fails → Return error dict, app continues
4. Dashboard shows appropriate status
5. Predictions are paused (not logged)
6. App keeps running, waiting for sources to recover

Verified by test:
• Ran test_failover_and_csv.py
• Result: "App handles failures gracefully - NO CRASH ✅"
• NSE API worked in test (got price 25178.65)
• App continued running even when simulating failures
