FILE: ai_engine_consensus.py

CHANGES:
✓ Modified get_consensus_prediction() to try XGBoost first
✓ Added _extract_xgb_features() to convert indicator summary to features
✓ Added _calculate_xgb_consensus() for weighted consensus
✓ Added _direction_to_text() and _text_to_direction() converters
✓ XGBoost gets 60% weight, Groq 20%, Gemini 20%
✓ Falls back to Dual-AI (50/50) if XGBoost not available

PREDICTION FLOW:
1. Try XGBoost prediction (if model exists)
2. Call Groq and Gemini APIs in parallel
3. Calculate weighted consensus:
   - All agree → STRONG consensus (+10% confidence)
   - XGB agrees with final → MODERATE consensus
   - XGB disagrees → WEAK consensus (-15% confidence)
4. Return final prediction with metadata

FALLBACK BEHAVIOR:
• If xgb_model.pkl doesn't exist → Use Dual-AI consensus
• If XGBoost prediction fails → Use Dual-AI consensus
• If both LLMs fail → Use XGBoost only
• If all fail → Use rule-based prediction
